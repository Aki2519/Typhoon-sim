# 台风路径模拟系统

热带气旋历史路径回放与统计分析工具。

## 运行

```bash
pip install -r requirements.txt
python main.py
```

## 操作

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `H` | 切换模式（正常 / 季节 / 编辑） |
| `[` `]` | 上一个 / 下一个台风 |
| `K` | 强度折线图 / ACE 图表 |
| `T` | 台风列表 |
| `J` | 历年统计（路径对比 / 热力图 / 路径长度） |
| `←` `→` | 减速 / 加速 |
| `X` | 速度重置 |
| `F12` | 窗口置顶 |
| `I` | 编辑模式新建台风 |
| 右键拖拽 | 平移地图 |
| 滚轮 | 缩放地图 |
