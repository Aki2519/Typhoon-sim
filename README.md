# 台风路径模拟系统

热带气旋历史路径回放与统计分析工具。

## 运行

```
pip install -r requirements.txt
python main.py
```

## 操作

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `H` | 切换模式（正常 / 季节 / 编辑） |
| `[` `]` | 上一个 / 下一个台风 |
| `K` | 强度折线图 / ACE 图表 |
| `T` | 台风列表 |
| `J` | 历年统计（路径对比 / 热力图 / 路径长度） |
| `←` `→` | 减速 / 加速 |
| `X` | 速度重置 |
| `F12` | 窗口置顶 |
| `I` | 编辑模式新建台风 |
| 右键拖拽 | 平移地图 |
| 滚轮 | 缩放地图 |

## 脚本格式示例

```
#测试脚本(简介)
/初始左下角顶点;绘制区域宽度
[5(在这里的停留时间,单位为秒,选填)
=2(移动到下一个绘制区域所需的时间,单位为秒,选填)
/第二个左下角顶点;绘制区域宽度
[[2026-04-12-00z(移动到这里的日期,其中最后的小时可以不填写,其他必填)
==2026-04-15(移出这里的日期,选填)
/第三个左下角顶点;绘制区域宽度
[[2026-04-16-00z(移动到这里的日期,其中最后的小时可以不填写,其他必填)
[5(在这里的停留时间,单位为秒,选填)
...
/最后一个左下角顶点;绘制区域宽度
[-1(永久停留)
=

运行脚本的按钮放在功能栏右边,名为"脚本"
弹出界面中列出所有脚本,按文件名排序,用小字展示简介.
绘制的高根据屏幕绘制区域的长宽比自己计算.
脚本运行过程中不能用鼠标拖动,但可以暂停,暂停时脚本也暂停计时.
用日期来移动时,镜头移动速度取决于前后的时间差及游戏速度.
需要注意的是,如果后面的日期比前面的日期提前,则可以填写第三个符号,这个符号只能是"=",这时游戏正常移动镜头过去,移过去的一瞬间,时间跳跃到后面的日期.
前面的符号必须填写,后面是空的就视为0s或2000-01-01-00z.
```
